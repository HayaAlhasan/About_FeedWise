Abstract

FeedWise transforms traditional academic feedback analysis into an intelligent AI-powered process.

FeedWise Automated Academic Feedback Summarizer Project is an automated system that handles extensive student feedback assessment in higher education. Universities collect student feedback through written comments to assess their teaching methods and enhance course content but face challenges because the unstructured data requires extended time for assessment which leads to unreliable results. The system automatically processes feedback through Natural Language Processing (NLP) tools that handle all steps from text preprocessing to sentiment detection and summary creation. The system organizes feedback into three distinct categories while it discovers essential topics and common patterns throughout the evaluation. The system provides academic decision-making support through structured insights which enable users to compare multiple courses and instructors. The system uses Python together with modern NLP libraries that include transformer-based models to achieve better accuracy. The solution successfully decreases the work required for manual tasks while it improves the process of finding valuable information from extensive datasets according to experimental evidence. The project develops a tool that educational data analysis researchers can use to create educational data analysis solutions which drive teaching and learning process improvements.

English Pitch

FeedWise is an AI-powered academic feedback summarizer designed to help universities understand student feedback faster and more accurately. Instead of manually reading hundreds of course evaluation comments, FeedWise uses Natural Language Processing, sentiment analysis, and transformer-based models to classify opinions, detect recurring issues, and generate clear summaries. The platform provides visual analytics that support instructors and administrators in making data-driven improvements to teaching quality. With an achieved accuracy of approximately 88%, FeedWise transforms raw student feedback into meaningful academic insights.
